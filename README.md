# ecommerce_automation
Overview

This project contains automation test scripts for core user authentication flows of an E-commerce application.
The automation focuses on validating User Registration and User Login functionality to ensure that the application behaves correctly for different user scenarios.

The goal of this project is to demonstrate automated UI testing for an E-commerce platform using modern automation tools and best testing practices.

Features Automated

The following user flows are automated:

User Registration

Register a new user with valid details

Validate successful account creation

Validate mandatory field validations

Validate error messages for invalid inputs

User Login

Login with valid credentials

Login with invalid credentials

Validate error messages

Verify successful user authentication
